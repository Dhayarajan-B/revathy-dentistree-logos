WHAT THE FILES ARE
------------------
Each concept has 4 files:
  NN-name.png        -> colour preview (open like a photo)
  NN-name.svg        -> colour vector master (give THIS to printers / sign-makers / designers)
  NN-name-mono.png   -> black-and-white preview
  NN-name-mono.svg   -> black-and-white vector master (for stamps, embossing, 1-colour print)

SVG files scale to any size without blurring — from favicon to billboard.
